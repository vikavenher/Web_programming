Головна сторінка Новини

    Варіант №1 - Сайт відображення новин

    - Навігація - шапка сайту

  • - Головна сторінка
  • - Новини
  • - Завдання
  • - Головна сторінка - перехід на index.html
  • - Новини - відображення списку новин
  • - Завдання - відображення варіанту завдання студента
  • - Новини у списку відображаються по 8 шт.
  • - Новина має заголовок, автора, час публікації та опис
  • - Є можливість відобразити ще 8 і т.д. за допомогою кнопки "показати більше новин"
  • - Опис новити приховано за промовчанням
  • - При виборі новини вона розгортається, відображаючи опис новини
  • - Загальна кількість новин - 28
  • - Дані зберігаються у папці /data
  • - Інформація про список новин зберігається у файлі data.txt
  • - Стиль оформлення сайту - на вибір студента
  • - Для верстки сайту використовується Bootstrap_5
  • - Для роботи з Ajax використовується jQuery_3
  • - Bootstrap та jQuery підключаються за допомогою CDN
